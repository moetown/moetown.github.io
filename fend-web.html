The FEND app landing page

June, 2018

The idea behind FEND is to engage youth into learning about opioids through an app using gamification to reengage, spur and motivate the user into continious learning. This is by the user achieved through answering questions, watching videos and infographics from motivational personalities backed by research.

I designed 2 different versions of the website as the app had two different versions released. The aim was to crate a look and feel that was matching the app's playful brand and direct the users from the FEND shopify page to the appstore via this landing page.
